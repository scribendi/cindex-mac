Adding spelling dictionaries
 

Cindex employs the spell checker Hunspell, which is widely used in public domain software such as Firefox, Thunderbird, and Google Chrome. Dictionaries for many languages, and some dictionaries containing specialized terms, are freely available.

To enable Cindex to use additional dictionaries, you must install them in the correct place. Follow these steps to find and install a dictionary:

•

For information on finding and downloading the dictionary you want, look here.

•

Append the extension ".zip", to the file you have downloaded, then unzip the file.

•

Inside the unzipped folder (or contained in a folder within it) will be a file that has the extension .dic and another (with the same name) that has the extension .aff. The files are named using a standard convention for representing languages and country variants. For example: en_US.aff and en_US.dic (US English); en_AU.aff and en_AU.dic (Australian English); nl_NL.aff and nl_NL.dic (Dutch).

•

Copy or drag the .aff file and the .dic file to this folder:
   Macintosh HD:Library:Spelling

If your disk is not named Macintosh HD, substitute the name in the instructions above.

If Cindex is active when you install a dictionary, you will need to restart Cindex to make the new dictionary available.
